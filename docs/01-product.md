# 01 — Продукт

## Видение

USTOZ — маркетплейс живых онлайн-занятий для Узбекистана. Ученик находит специалиста по любому направлению, бронирует слот, оплачивает картой (Payme/Click/Uzum) и проходит урок **внутри платформы**: видео, интерактивная доска, чат, файлы, домашние задания. Платформа удерживает обе стороны внутри: вся коммуникация и оплата — только через USTOZ.

## Роли

1. **Ученик (student)** — ищет, бронирует, оплачивает, учится, оставляет отзывы.
2. **Преподаватель / специалист (teacher)** — создаёт профиль, ставит расписание и цены, проводит уроки, получает деньги на внутренний кошелёк, выводит по запросу.
3. **Админ (admin)** — модерация, выплаты, категории, метрики, настройки (отдельная панель).

Один аккаунт может иметь обе роли (ученик ↔ преподаватель), переключение в профиле. Регистрация преподавателя свободная, без проверки документов. Поле `is_verified` (бейдж «Проверенный специалист») заложено в схему, но в MVP выключено — включается позже решением админа.

## Категории и предметы (seed-данные, управляются из админки)

| Категория | Примеры предметов |
|---|---|
| Языки | Английский, русский, узбекский, арабский, корейский, немецкий, китайский, французский, турецкий |
| Школьные предметы | Математика, физика, химия, биология, история, подготовка к ДТМ/IELTS/SAT |
| IT и программирование | Python, веб-разработка, мобильная разработка, дизайн, Excel, 1C |
| Психология и коучинг | Консультации психолога, семейный психолог, карьерный коуч |
| Бизнес и финансы | Бухгалтерия, маркетинг, продажи, налоги |
| Творчество и музыка | Гитара, фортепиано, вокал, рисование, фотография |

Предмет принадлежит категории. Преподаватель привязывает к себе 1+ предметов и для каждого задаёт цены.

## Бизнес-модель

**Комиссии с уроков нет.** Деньги ученика за урок зачисляются на внутренний кошелёк преподавателя в полном объёме (за вычетом эквайринга платёжной системы, ~1–2%, удерживается из суммы — отражается в транзакции).

Платформа зарабатывает на **подписке преподавателей**:

| | FREE | PRO |
|---|---|---|
| Цена | 0 | 149 000 UZS/мес *(плейсхолдер, настраивается в админке)* |
| Проведённых уроков в месяц | до 10 | безлимит |
| Предметов в профиле | 1 | до 5 |
| Позиция в поиске | стандартная | приоритетная (boost) |
| Бейдж PRO на профиле | — | ✓ |
| Видео-презентация в профиле | ✓ | ✓ |
| Аналитика (просмотры, конверсия, доход) | базовая | полная |

- При достижении лимита FREE новые бронирования к преподавателю блокируются до конца месяца → paywall «Перейти на PRO».
- Оплата PRO в MVP — вручную раз в месяц через Payme/Click (преподаватель получает напоминание за 3 дня). Автосписание по токену карты (Payme Subscribe API) — фаза 7.
- Все лимиты/цены — параметры в таблице `app_settings`, меняются из админки без релиза.

## Цены, пробные уроки, пакеты

- Преподаватель задаёт цену за урок **30 / 60 / 90 минут** по каждому предмету.
- **Бесплатный пробный урок 20 минут** — опция, преподаватель включает сам. Один бесплатный пробный на пару «ученик–преподаватель».
- **Платный пробный со скидкой** — преподаватель задаёт % скидки на первый полный урок.
- **Пакеты 5 / 10 / 20 уроков** — преподаватель задаёт скидку на пакет (например 5%, 10%, 15%). Ученик покупает пакет → урокам списываются «кредиты пакета» при бронировании. Срок жизни пакета: 6 месяцев (настройка).

## Отмены и переносы (политика по умолчанию, настраивается)

- Отмена учеником **≥ 12 ч** до урока — возврат 100% на **внутренний баланс ученика** (тратится на следующие бронирования; вывод живых денег — вручную через поддержку/админку).
- Отмена **< 12 ч** — без возврата (деньги уходят преподавателю).
- Отмена преподавателем в любой момент — возврат ученику 100% на баланс + штрафной флаг преподавателю (3 отмены за 30 дней → понижение в выдаче, уведомление админу).
- Перенос по согласию обеих сторон в чате → кнопка «Предложить перенос» (новый слот, подтверждение второй стороной).
- No-show ученика (не зашёл 15 мин) — урок считается проведённым. No-show преподавателя — возврат ученику + штрафной флаг.

## Удержание внутри платформы (anti-bypass)

- В чате до и между уроками **маскируются** телефоны, ссылки, @-хэндлы Telegram/Instagram (regex на клиенте + сервере, замена на `[скрыто]`). Флаг `chat_masking_enabled` в настройках.
- В профиле преподавателя запрещены контакты (модерация текста профиля: автоскрытие при срабатывании regex + очередь на проверку в админке).
- Уроки проходят только во встроенной видеокомнате; ссылка на комнату генерируется за 10 минут до начала и работает только для двух участников брони.

## Ключевые пользовательские сценарии

### Ученик
1. Регистрация: телефон → SMS-код → имя → выбор интересующих категорий (для персонализации главной).
2. Поиск: каталог → фильтры (категория, предмет, цена от/до, рейтинг, язык преподавания uz/ru/en, наличие бесплатного пробного, время доступности) + сортировка (рекомендуемые/цена/рейтинг).
3. Профиль преподавателя: фото, видео-презентация, био, предметы и цены, рейтинг и отзывы, календарь свободных слотов.
4. Бронирование: выбрать предмет → длительность → слот → способ (разовый / пробный / из пакета) → оплата Payme/Click/Uzum → подтверждение + событие в «Мои уроки» + напоминания (push за 24ч и за 1ч, SMS за 1ч).
5. Урок: кнопка «Войти в урок» активна за 10 мин; видеокомната с доской, чатом, файлами; по окончании — экран оценки (1–5 звёзд + текст).
6. После: домашка от преподавателя → выполнение → XP/стрики → прогресс в профиле.

### Преподаватель
1. Регистрация → переключение «Стать преподавателем» → анкета: фото, био (uz/ru), опыт, видео-презентация (до 90 сек, загрузка в Storage), предметы и цены, включение пробных.
2. Расписание: недельная сетка доступности (повторяющиеся слоты) + исключения (отпуск/праздники). Слоты нарезаются по 30 мин.
3. Заявки приходят уже оплаченными (мгновенное подтверждение, без ручного аппрува — слот блокируется при оплате).
4. Проведение урока → автозачисление суммы на кошелёк после статуса `completed` (через 24 ч защитного холда).
5. Кошелёк: баланс, история транзакций, «Запросить выплату» (сумма + номер карты Humo/Uzcard) → статус → деньги (админ подтверждает).
6. PRO: paywall при лимите, страница тарифов, оплата, бейдж.

### Админ — см. `docs/06-admin-panel.md`.

## Геймификация (ученики)

- **XP**: +50 за проведённый урок, +20 за сданную домашку, +10 за выполненный тест, +5 за отзыв.
- **Стрик**: дни подряд с активностью (урок или домашка). Заморозка стрика — 1 шт/мес бесплатно.
- **Уровни**: Новичок → Ученик → Знаток → Мастер → Гуру (пороги XP в настройках).
- Прогресс-экран: XP, уровень, стрик, история занятий, «предметы в изучении».
- Лидерборд — фаза 7 (не MVP).

## Не входит в MVP (фиксируем, чтобы не расползтись)

Групповые уроки, запись уроков, лидерборд, международные карты, автосписание подписки, верификация документов, реферальная программа, блог, web-версия видеокомнаты на мобильном браузере (мобильный веб ведёт в приложение).
