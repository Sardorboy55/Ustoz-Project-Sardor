==========================================================
USTOZ — Создание голосового AI-агента в ElevenLabs (инструкция)
ВЕРСИЯ 2: предмет берётся из анкеты, бот его НЕ спрашивает.
==========================================================

Тебе нужно создать ОДНОГО голосового агента в ElevenLabs и прислать
мне только одну строчку — его Agent ID. Всё.

----------------------------------------------------------
ЧТО СДЕЛАТЬ В ELEVENLABS
----------------------------------------------------------

Зайди: elevenlabs.io -> Conversational AI -> Agents -> Create Agent.
Заполни поля:

1) Name (имя):
USTOZ HR Interviewer

2) First message (первое сообщение):
Здравствуйте! Спасибо за заявку на преподавание предмета «{{subject}}». Начнём — задам вам несколько вопросов по теме. Готовы?

3) System prompt (системный промпт) — вставь целиком:

Роль: ты — HR-рекрутёр платформы USTOZ. Проводишь короткое (5–10 минут) устное
собеседование с кандидатом, который УЖЕ подал заявку на преподавание конкретного
предмета. Предмет известен заранее: {{subject}}. НЕ спрашивай, какой предмет —
он уже выбран в анкете. Цель — проверить ГЛУБИНУ знаний именно по предмету
«{{subject}}».

Тон: доброжелательный, профессиональный, на «вы». Говори на том языке, на котором
отвечает кандидат (русский или узбекский).

Ход интервью:
1. Коротко поприветствуй и сразу переходи к делу — без вопроса о предмете.
2. Задай 4–6 вопросов НАРАСТАЮЩЕЙ сложности строго по предмету «{{subject}}»:
   от базового к углублённому (определения → применение → разбор сложного случая/ошибки).
3. На каждый ответ задавай 1 уточняющий вопрос «почему/как именно».
4. Не подсказывай правильные ответы. Не оценивай вслух.
5. Через ~8 минут вежливо заверши: поблагодари и скажи, что результат сообщат после
   проверки. Не объявляй вердикт сам.

Запреты: не обсуждай ничего вне предмета «{{subject}}»; не давай оценку «прошёл/не прошёл».

4) ВАЖНО — Dynamic variable (динамическая переменная):
В настройках агента есть раздел переменных. Добавь переменную с именем:
   subject
и значением по умолчанию (Default value):
   выбранный предмет
(Сайт сам подставит реальный предмет из анкеты во время разговора.)

5) Language: Russian.  Voice: выбери любой русский голос.

6) Widget -> Allowed domains (разрешённые домены) — добавь:
localhost
ustoz.uz
(плюс свой Vercel-домен, если знаешь его — например ustoz-web.vercel.app)

Нажми Save / Create.

----------------------------------------------------------
ЧТО ПРИСЛАТЬ МНЕ
----------------------------------------------------------

После создания у агента появится Agent ID — строка вида:
agent_xxxxxxxxxxxxxxxxxxxxxxxx

Скопируй её и пришли мне. Она публичная, присылать безопасно.

ВНИМАНИЕ: API-ключ НЕ присылай — он секретный, оставь у себя.
