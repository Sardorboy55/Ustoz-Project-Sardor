# Агенты IBILIM

Именные субагенты-специалисты для Claude Code. Каждый ведёт свою зону.

| Агент | `subagent_type` | Зона |
|---|---|---|
| **Jony** | `jony-bugs` | Баги, краши, регрессии, рантайм-ошибки приложения / APK |
| **Hanna (Анна)** | `hanna-auth` | Регистрация и вход: Google, Telegram, Supabase Auth, сеть авторизации |
| **John** | `john` | Вёрстка, дизайн, соответствие сайту, overflow, цвета/отступы/кнопки |
| **Alisa** | `alisa` | Сторона преподавателей: стать учителем, кабинет, профиль, предметы/цены |
| **Max** | `max` | Сторона учеников/клиентов: каталог, бронь, уроки, оплата, кошелёк, чат |
| **Vera** | `vera-voice` | Голосовой ИИ-агент собеседования (ElevenLabs): обрывы/перебивание, микрофон в WebView, conversation_id |

Вызов: по имени («John, поправь…») или автоделегирование по теме.

## Где лежат / как подключены
Реальные файлы — здесь, в `Ustoz/agents/` (в git, общие для всех маков).
Claude Code грузит агентов из `.claude/agents/`, поэтому там стоит **симлинк** на эту папку:
```
ln -s ../agents .claude/agents     # выполнить один раз на каждом маке (.claude в .gitignore)
```
После создания/изменения агентов — **перезапустить Claude Code** (список агентов читается при старте).

## ⚠️ Проверка перед сборкой APK / деплоем (обязательно)
Перед КАЖДОЙ сборкой APK или деплоем — прогнать проверку через агентов
(параллельно) и собирать только если ВСЕ подтвердили «ок»:

1. **jony-bugs** — всегда: нет багов/регрессий/крашей в изменённом коде, сборка скомпилируется.
2. **Профильные агенты по затронутым зонам:**
   - менялся вход/сеть авторизации → **hanna-auth** (`SUPABASE_URL=ibilim.uz/supa`, пин, `INTERNET` permission);
   - менялась вёрстка/экраны → **john** (нет overflow, бренд-цвета, совпадает с сайтом);
   - менялась сторона преподавателей → **alisa**;
   - менялась сторона учеников/оплата → **max**;
   - менялся голосовой агент собеседования (interview-agent.html / WebView) → **vera-voice**.

Если кто-то нашёл проблему — **сначала почини, потом пересобирай**. Не отдавать
APK с непройденной проверкой.
