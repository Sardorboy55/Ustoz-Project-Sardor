# Агенты IBILIM

Именные субагенты-специалисты для Claude Code. Каждый ведёт свою зону.

| Агент | Имя (`subagent_type`) | Зона |
|---|---|---|
| **Jony** | `jony-bugs` | Баги, краши, регрессии, рантайм-ошибки мобильного приложения / APK |
| **Hanna (Анна)** | `hanna-auth` | Регистрация и вход: Google, Telegram, Supabase Auth, сеть авторизации |
| **UI/UX** | `ui-ux` | Вёрстка, дизайн, соответствие сайту, overflow, цвета/отступы/кнопки |

Вызов: по имени («Jony, проверь…») или автоделегирование по теме.

## Где лежат / как подключены
Реальные файлы — здесь, в `Ustoz/agents/` (в git, общие для всех маков).
Claude Code грузит агентов из `.claude/agents/`, поэтому там стоит **симлинк** на эту папку:
```
ln -s ../agents .claude/agents     # выполнить один раз на каждом маке (.claude в .gitignore)
```
После создания/изменения агентов — **перезапустить Claude Code** (список агентов читается при старте).

## ⚠️ Проверка перед сборкой APK / деплоем (обязательно)
Перед КАЖДОЙ сборкой APK или деплоем — прогнать проверку доменов через агентов
(параллельно), и собирать только если ВСЕ подтвердили «ок»:

1. **jony-bugs** — нет багов/регрессий/крашей в изменённом коде, сборка скомпилируется.
2. **hanna-auth** — вход цел: `SUPABASE_URL=ibilim.uz/supa`, пин на `ibilim.uz`,
   `INTERNET` permission в release-манифесте, логика `auth_screen`/`auth_repository`.
3. **ui-ux** — вёрстка корректна: нет overflow, бренд-цвета/отступы на месте,
   совпадает с сайтом где требуется.

Если кто-то нашёл проблему — **сначала почини, потом пересобирай**. Не отдавать
APK с непройденной проверкой.
