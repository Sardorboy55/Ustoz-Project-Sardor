# USTOZ — маркетплейс преподавателей и специалистов

> **Рабочее название — USTOZ** («наставник» на узбекском). Можно заменить — везде в документах используется как плейсхолдер.

Онлайн-платформа для Узбекистана: поиск, бронирование и проведение уроков/консультаций с преподавателями и специалистами **внутри платформы** — как italki/Preply, но для всех категорий: языки, IT, школьные предметы, психологи, мастера.

## Ключевые решения (зафиксированы)

- **Рынок:** Узбекистан. Языки интерфейса: узбекский + русский. Часовой пояс: Asia/Tashkent. Валюта: UZS.
- **MVP:** сразу все категории специалистов.
- **Клиенты:** Flutter (iOS + Android), Next.js (сайт, SEO-профили), админка на Next.js.
- **Бэкенд:** Supabase (PostgreSQL, Auth, Storage, Realtime, Edge Functions).
- **Видео:** Agora на старте (10 000 бесплатных минут/мес) → миграция на self-hosted LiveKit при росте. Код пишется через слой абстракции `VideoProvider`.
- **Вход:** телефон + SMS-код (Eskiz.uz).
- **Монетизация:** подписка преподавателей Free / PRO. Комиссии с уроков нет — ученик платит, деньги падают на внутренний кошелёк преподавателя.
- **Выплаты:** по запросу преподавателя, подтверждение в админке.
- **Оплата:** Payme, Click, Uzum (MVP) → международные карты через OCTO/Payze (фаза 2).
- **Пробные уроки:** бесплатный 20 мин + платный со скидкой. Пакеты 5/10/20 уроков со скидкой.
- **Регистрация преподавателей:** свободная, без проверки документов. В админке — бан и (на будущее) бейдж «Проверенный специалист».
- **Интерактив:** видеоурок в приложении, интерактивная доска, обмен файлами, чат с маскировкой контактов, домашние задания, тесты, геймификация (XP, стрики, уровни).

## Что нужно завести до старта разработки (аккаунты)

1. **Supabase** — новый проект (бесплатный тариф для разработки).
2. **Agora.io** — аккаунт, создать проект, получить App ID + App Certificate.
3. **Eskiz.uz** — аккаунт для SMS (нужен договор для отправки на реальные номера; для тестов есть тестовый режим).
4. **Firebase** — проект для push-уведомлений (FCM) + ключи для iOS/Android.
5. **Payme Business** — заявка на мерчанта (мерчант-ID + ключ). Параллельно: **Click** и **Uzum** мерчант-аккаунты. *Заявки рассматриваются 1–2 недели — подать сразу, интеграция кодом идёт параллельно на тестовых кредах.*
6. **Apple Developer** ($99/год) и **Google Play Console** ($25) — для публикации.
7. Домен (например `ustoz.uz`) + поддомены: `app.` (веб-кабинет), `admin.`, `api.` (если понадобится).

## Как запустить работу с Claude Code

```bash
# 1. Создать репозиторий и положить туда этот пакет
mkdir ustoz && cd ustoz
# скопировать CLAUDE.md в корень, папку docs/ рядом

# 2. Запустить Claude Code
claude

# 3. Первая команда:
# "Прочитай CLAUDE.md и docs/, начни Фазу 0 из docs/08-roadmap.md"
```

Claude Code будет идти по фазам из `docs/08-roadmap.md`. Каждая фаза имеет критерии приёмки — проверяй их перед переходом к следующей.
